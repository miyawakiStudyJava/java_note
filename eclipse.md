コピペと削除

|ショートカット|キー|補足|
|---|---|---|
|全選択	|Ctrl + A||
|コピー	|Ctrl + C	|任意の箇所を選択|
|ペースト	|Ctrl + V	|任意の箇所を選択|
|カット	|Ctrl + X	|任意の箇所を選択|
|★行コピペ	|Ctrl + Alt + ↓	|１行下にペーストされる|
|★行削除	|Ctrl + D	||
|キャレットのから行末まで削除	|Ctrl + Shift + Delete	||
